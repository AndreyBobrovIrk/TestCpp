# TestCpp

Решение тестовых задач на вакансию "Программист C++"

Решение представляет собой 5 консольных приложений написанных на C++.
Названия проектов соответствуют номерам заданий из файла (Task1, Task2 и т.д.).

В качестве среды разработки использована MS Visual Studio 2017.
